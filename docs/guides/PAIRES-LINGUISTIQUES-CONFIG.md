# Configuration des paires linguistiques par division

## Question : Quelles paires appliquer ?

Vous avez 65 traducteurs sans paires linguistiques :
- **44** dans Traduction anglaise (1 & 2)
- **21** dans Multilingue

### Options possibles :

#### 1. Traduction anglaise 1 & 2 (44 traducteurs)
Choisissez :
- [ ] FR → EN (français vers anglais uniquement)
- [ ] EN → FR (anglais vers français uniquement)  
- [ ] FR ↔ EN (bidirectionnel : FR→EN + EN→FR)

#### 2. Multilingue (21 traducteurs)
Choisissez les paires à appliquer :
- [ ] EN ↔ FR (anglais-français bidirectionnel)
- [ ] ES ↔ FR (espagnol-français bidirectionnel)
- [ ] DE ↔ FR (allemand-français bidirectionnel)
- [ ] Autres langues ?
